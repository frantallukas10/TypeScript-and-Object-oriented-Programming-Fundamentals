# TypeScript and Object-oriented Programming Fundamentals

How to install TypeScript globally

```js
npm install -g typescript
```

TypeScript can be compile to Javascript with command:

```js
tsc fileName.ts
```

## Benefits:

- Strong typing
- Object-oriented features
- Compile-time errors
- Great tooling
